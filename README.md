AI-Driven-Development
=====================

AI を活用した開発プロセスの各フェーズをまとめるリポジトリです。要件定義から設計、実装までの成果物と補助資料を段階的に整理します。
今回は、CodeX × Next.jsの開発として話を進めますが、他の言語への拡張も可能であると考えています。

プロジェクト概要
----------------
- AI 駆動開発の「基盤」を作るための知見を蓄積し、再利用できる形で整理する。
- 要件定義から実装・運用まで、各フェーズで得たベストプラクティスや失敗談を記録する。
- プロンプト設計やエージェント設計、設計図や開発手順を「いつでも参照できるナレッジベース」として育てる。

ディレクトリ構成
----------------
- `1-要件定義` : 目的・スコープ・ユーザーストーリーなどの要件整理。
- `2-設計` : アーキテクチャやデータ設計、技術選定の記録。
- `3-AGENTS.mdの作成` : エージェント設計やプロンプトの設計メモ。
- `4-開発` : 実装フェーズのコード・手順・運用方針。
- `diagram` : 図式や可視化資料。

進め方のメモ
------------
1. まず `1-要件定義` で課題・ゴール・制約を明文化する。
2. 設計方針は `2-設計` に集約し、決定の背景を残す。
3. エージェント関連の設定やプロンプトは `3-AGENTS.mdの作成` にまとめる。
4. 実装・検証の手順や成果物は `4-開発` に追加する。
5. 必要に応じて `diagram` 配下で図を更新し、各フェーズの README から参照する。
