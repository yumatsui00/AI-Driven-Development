開発基盤（3）ディレクトリ概要
=============================

目的
----
AI 駆動開発の基盤を整えるための設計・運用メモを集約する。ここでは、AGENTS.md の作成、CI/CD セットアップ、AI レビュー基盤の 3 つの柱に分けて、再利用可能なノウハウと手順を残す。

ディレクトリ構成
----------------
- `Agents.mdの作成/` : プロジェクト全体参照ファイル（AGENTS.md）の構成と記載テンプレート。
- `CI,CDのセットアップ/` : CI/CD の設計方針、ワークフロー例、チェック項目。
- `AIレビュー基盤セットアップ/` : AI を用いたコードレビューや品質チェックの仕組みづくり。

運用ルール
----------
- 機能設計が終わったら、各ディレクトリの README を起点に手順を固め、都度更新する。
- 共通化できる設定やテンプレートはリポジトリに残し、次のプロジェクトで再利用できるようにする。
